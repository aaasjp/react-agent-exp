# 安装和启动指南

## 安装步骤

按顺序执行以下命令安装项目依赖：

```bash
# 1. 安装项目本身（开发模式）
python -m pip install -e .

# 2. 安装开发依赖
python -m pip install ".[dev]"

# 3. 安装 LangGraph CLI（内存模式）
python -m pip install "langgraph-cli[inmem]"
```

## 启动命令

### 标准启动（推荐）

```bash
langgraph dev
```

### 允许阻塞操作启动

如果遇到阻塞操作错误（BlockingError），可以使用以下命令启动：

```bash
langgraph dev --allow-blocking
```

**注意**：`--allow-blocking` 标志会绕过阻塞检查，允许同步阻塞操作，但可能会影响性能。建议优先修复代码中的阻塞操作，使用异步方式处理文件系统操作。

## 环境变量方式（可选）

也可以通过设置环境变量来允许阻塞操作：

```bash
export BG_JOB_ISOLATED_LOOPS=true
langgraph dev
```

## 快速开始

完整安装和启动流程：

```bash
# 安装
python -m pip install -e .
python -m pip install ".[dev]"
python -m pip install "langgraph-cli[inmem]"

# 启动（如果遇到阻塞错误，使用 --allow-blocking）
langgraph dev --allow-blocking
```
